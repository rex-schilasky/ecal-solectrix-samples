# Solectrix / eCAL image publication samples

## How to build

1. Install cmake and git / git-for-windows

2. CMake the project
	```console
	mkdir _build
	cd _build
	cmake ..
	```

3. Build the project
	- Linux: `make`
	- Windows: Open `_build\ecal_solectrix_samples.sln` with Visual Studio and build the example project
